About the project

Frene-Urbain

This was my first year group project as an undergraduate.
This is an online property selling website where both sellers and buyers can register and interact.
Property owners can list the properties they have for sale and buyers can search and view them.
If a buyer wishes to reserve a property for checking (before purchase) they can do so through the website.
I was responsible for developing the buyer user interface, buyer account creation and buyer database creation and connectivity.
